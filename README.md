# Helsinki Traffic Analysis

This project uses the Statistical API for traffic measurement data provided by the City of Helsinki.

The dataset includes traffic volume, average speeds, and vehicle counts collected for different means of transportation — vehicles, pedestrians, and cyclists. Data is gathered from sources such as LAM stations in the Helsinki region and pedestrian counting devices, and is available in both JSON and CSV formats via a REST API.

## Data Source

- [Helsinki Open Data](https://avoindata.suomi.fi/data/en_GB/dataset/helsingin-liikennemittausten-tilastorajapinta)
- [API Swagger UI](https://lidotiku.api.hel.fi/swagger#/api/api_observations_aggregate_list)
